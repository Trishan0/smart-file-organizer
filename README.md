# 📂 smart-file-organizer
A simple and efficient Python script to organize files in a directory by moving them into categorized folders based on their file extensions. Perfect for cleaning up messy directories!

# ✨ Features

📁 Organize Files: Automatically sorts files into folders like Videos, Pictures, Documents, Music, and Archives.

📍 Flexible Path Selection: Organize files in the current directory or specify a custom directory.

🖥️ Easy to Use: Simple command-line interface with clear prompts.

🚀 Lightweight: No dependencies other than Python's standard library.
